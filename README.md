# The Click Race - TRÓPICA (Mercado Ads Regional Tour)

Desarrollo de una experiencia interactiva de alta energía diseñada para la gira regional de **Mercado Ads**. El sistema gestiona una competencia de clics entre tótems digitales con servidores locales independientes de la conexión a internet.

## 🚀 Resumen del Proyecto
El objetivo es crear una infraestructura técnica independiente, resiliente y con cero interrupciones para eventos en vivo en múltiples mercados internacionales.

### Características Principales
* **Arquitectura Offline-First:** Servidor local en cada tótem (Node.js + SQLite) para que la lógica y los datos de participantes no dependan de la nube.
* **Rendimiento Visual:** Uso de **GSAP** (GreenSock Animation Platform) para animaciones fluidas y profesionales con aceleración por hardware.
* **Curvado de Dificultad:** Lógica de juego diseñada para ajustar la dificultad de la experiencia táctil.
* **Localización Regional:** Implementación técnica de variantes en Español e Inglés (MX, AR, CL, CO, UY) y Portugués (BR).

## 🛠️ Especificaciones Técnicas
### Requerimientos de Hardware
Para el correcto funcionamiento del software en Modo Kiosk, el tótem debe contar con:
* **Procesador:** Intel i5 (8.ª gen o superior).
* **RAM:** 8 GB.
* **Almacenamiento:** 64 GB SSD.
* **Pantalla:** Interfaz táctil vertical (9:16).

### Stack Tecnológico
* **Backend:** Node.js.
* **Base de Datos:** SQLite.
* **Frontend:** Web App para navegadores Chromium.
* **Animaciones:** GSAP.
