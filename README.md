# The Click Race - TRÓPICA (Mercado Ads Regional Tour)

![Status](https://img.shields.io/badge/Status-Project_Proposal-blue)
![Platform](https://img.shields.io/badge/Architecture-Offline--First-orange)
![Markets](https://img.shields.io/badge/Markets-6_Regions-green)

[cite_start]Desarrollo de una experiencia interactiva de alta energía diseñada para la gira regional de **Mercado Ads**[cite: 11]. [cite_start]El sistema gestiona una competencia de clics entre tótems digitales con servidores locales independientes de la conexión a internet[cite: 12].

## 🚀 Descripción del Proyecto
[cite_start]El objetivo es crear una infraestructura técnica independiente, resiliente y con cero interrupciones para eventos en vivo[cite: 20].

### Características Principales
* [cite_start]**Arquitectura Offline-First:** Servidor local en cada tótem ($Node.js + SQLite$) para que la lógica y los datos no dependan de la nube[cite: 15].
* [cite_start]**Rendimiento Visual:** Uso de **GSAP** (GreenSock Animation Platform) para animaciones fluidas con aceleración por hardware[cite: 17].
* [cite_start]**Curvado de Dificultad:** Lógica programada para adaptar la dificultad de la competencia en tiempo real[cite: 24].
* [cite_start]**Localización:** Soporte técnico para variantes en Español (MX, AR, CL, CO, UY) y Portugués (BR)[cite: 21, 23].

## 🛠️ Especificaciones Técnicas
### Hardware Requerido (Tótems)
[cite_start]Para el correcto funcionamiento del Modo Kiosk, se requiere[cite: 18]:
* **Procesador:** i5 (8.ª gen)
* **RAM:** 8 GB
* **Almacenamiento:** 64 GB SSD
* **SO:** Navegadores basados en Chromium

### Stack Tecnológico
* [cite_start]**Backend:** Node.js.
* [cite_start]**Base de Datos:** SQLite[cite: 15].
* [cite_start]**Frontend:** Web App (9:16 vertical)[cite: 18, 24].
* [cite_start]**Animaciones:** GSAP[cite: 17].
